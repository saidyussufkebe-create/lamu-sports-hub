# 🏆 Lamu Sports Hub - Talent Exposure Platform

> Empowering footballers in Lamu County through visibility, recognition, and opportunity.

## 📋 Project Overview

**Lamu Sports Hub** is a comprehensive talent exposure platform designed to showcase football players from Lamu County. The platform bridges the gap between emerging athletes and scouts, enabling talent discovery, player exposure, and career advancement opportunities.

### 🎯 Mission
To provide Lamu County footballers with a dedicated platform to gain exposure, build their personal brand, and connect with scouts and coaches.

### 🌟 Core Features

#### 1. **Player Profiles**
- Comprehensive player statistics and achievements
- Career history and performance metrics
- Personal bio and contact information
- Player ratings and reviews from scouts

#### 2. **Highlight Uploads**
- Video upload functionality for performance highlights
- Clip management and organization
- Privacy controls and visibility settings
- Video moderation and quality standards

#### 3. **Scout/Coach View**
- Advanced filtering by position, age, height, weight
- Performance-based search capabilities
- Detailed player analysis tools
- Rating and comment system

#### 4. **Weekly Rankings**
- Dynamic leaderboard based on performance metrics
- Position-specific rankings
- Historical rank tracking
- Achievement badges and recognitions

#### 5. **Data Collection System**
- Reporter/Scout registration system
- Match data input forms
- Performance metrics tracking
- Automated data validation

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Git
- Firebase account
- Basic knowledge of React/Vue

### Installation

```bash
# Clone the repository
git clone https://github.com/saidyussufkebe-create/lamu-sports-hub.git
cd lamu-sports-hub

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Start development server
npm run dev
```
